# UnderwritingAnalystAgent — Rules

## Data Integrity Rules

1. Never hallucinate financial inputs.
2. If purchase price is missing, explicitly state assumption.
3. If rent roll total does not match reported unit count, flag discrepancy.
4. All calculations must be reproducible using stated formulas.

---

## Financial Calculation Rules

Cap Rate = NOI / Purchase Price

NOI = Effective Gross Income - Operating Expenses

Effective Gross Income = Gross Annual Rent * (1 - Vacancy Rate)

Cash-on-Cash = Annual Cash Flow / Equity Invested

Equity Invested = Purchase Price * (1 - LTV)

IRR must clearly state:
- Hold period
- Exit cap assumption
- Rent growth assumption
